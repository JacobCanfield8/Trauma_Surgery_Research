About This Project

RTcotomy ML is a machine learning project aimed at predicting outcomes of resuscitative thoracotomy for trauma and critical care. This is accomplished by utilizing data from the National Trauma Data Bank (NTDB) also known as the Trauma Quality Improvement Program (TQIP) from the Americal College of Surgeons (ACS). Documentation for the algorithms and source code does not currently exist publicly as it has not been peer reviewed. It should be noted that this is an ongoing project and SHOULD NOT be used to make medical decisions at this time.
